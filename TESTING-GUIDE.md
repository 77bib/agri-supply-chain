# دليل اختبار نظام عربة التسوق المخصصة للعملاء

## خطوات الاختبار السريع

### 1. تشغيل التطبيق
```bash
npm run dev
```
التطبيق يعمل على: http://localhost:3000

### 2. إنشاء حساب جديد
1. اذهب إلى http://localhost:3000/register
2. أنشئ حساب جديد
3. تحقق من تسجيل الدخول بنجاح

### 3. اختبار عربة التسوق المخصصة

#### الخطوة 1: الوصول إلى عربة التسوق
1. بعد تسجيل الدخول، انقر على اسمك في الأعلى
2. اختر "My Cart" من القائمة المنسدلة
3. يجب أن تظهر صفحة عربة التسوق المخصصة

#### الخطوة 2: إضافة منتجات
1. اذهب إلى صفحة المنتجات: http://localhost:3000/products
2. أضف منتج أو أكثر إلى عربة التسوق
3. عد إلى "My Cart" وتأكد من ظهور المنتجات

#### الخطوة 3: تعديل معلومات العميل
1. في صفحة "My Cart"، انقر على "تعديل" بجانب معلومات العميل
2. عدّل المعلومات (الاسم، الهاتف، العنوان)
3. احفظ التغييرات
4. تحقق من حفظ البيانات

#### الخطوة 4: اختبار الحفظ التلقائي
1. أضف منتجات إلى عربة التسوق
2. أغلق المتصفح تماماً
3. افتح المتصفح مرة أخرى
4. سجل الدخول
5. اذهب إلى "My Cart"
6. تحقق من استعادة المنتجات

#### الخطوة 5: اختبار تسجيل الخروج والدخول
1. أضف منتجات إلى عربة التسوق
2. سجل الخروج
3. سجل الدخول مرة أخرى
4. اذهب إلى "My Cart"
5. تحقق من استعادة عربة التسوق

### 4. اختبار الميزات المتقدمة

#### تعديل الكميات:
1. في "My Cart"، استخدم أزرار + و - لتعديل الكميات
2. تحقق من تحديث المجموع الكلي

#### إزالة منتجات:
1. انقر على أيقونة سلة المهملات بجانب المنتج
2. تحقق من إزالة المنتج وتحديث المجموع

#### تفريغ عربة التسوق:
1. انقر على "تفريغ عربة التسوق"
2. تحقق من إزالة جميع المنتجات

### 5. اختبار الأمان

#### بدون تسجيل دخول:
1. حاول الوصول إلى http://localhost:3000/customer-cart
2. يجب أن يتم توجيهك إلى صفحة تسجيل الدخول

#### مع توكن غير صالح:
1. احذف التوكن من localStorage
2. حاول الوصول إلى "My Cart"
3. يجب أن يتم توجيهك إلى تسجيل الدخول

## التحقق من قاعدة البيانات

### فحص البيانات في MongoDB:
```bash
# الاتصال بـ MongoDB
mongosh

# اختيار قاعدة البيانات
use agrichain

# عرض عربات التسوق
db.carts.find()

# عرض عربة تسوق لمستخدم محدد
db.carts.findOne({userId: "USER_ID_HERE"})
```

## اختبار API مباشرة

### جلب عربة التسوق:
```bash
curl -X GET http://localhost:3000/api/customer-cart \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"
```

### إضافة منتج:
```bash
curl -X POST http://localhost:3000/api/customer-cart/items \
  -H "Authorization: Bearer YOUR_JWT_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "productId": "test123",
    "name": "منتج تجريبي",
    "description": "وصف المنتج",
    "price": 25.00,
    "category": "اختبار",
    "stock": 10,
    "quantity": 2
  }'
```

- [ ] يتم حفظ المنتجات في قاعدة البيانات
- [ ] يتم استعادة عربة التسوق عند تسجيل الدخول مرة أخرى
- [ ] يمكن تعديل معلومات العميل وحفظها
- [ ] يتم تحديث المجموع عند تعديل الكميات
- [ ] يمكن إزالة المنتجات وتفريغ عربة التسوق

## استكشاف الأخطاء

### إذا لم تعمل عربة التسوق:
1. تحقق من تشغيل MongoDB
2. تحقق من console المتصفح للأخطاء
3. تحقق من logs الخادم
4. تحقق من صحة التوكن

### إذا لم تُحفظ البيانات:
1. تحقق من اتصال قاعدة البيانات
2. تحقق من صحة التوكن
3. تحقق من API endpoints

### إذا لم تُستعاد البيانات:
1. تحقق من وجود بيانات في قاعدة البيانات
2. تحقق من localStorage
3. تحقق من عملية تسجيل الدخول

## ملاحظات مهمة

- تأكد من تشغيل MongoDB قبل اختبار النظام
- تأكد من وجود بيانات منتجات في قاعدة البيانات
- النظام يحفظ البيانات تلقائياً عند أي تغيير
- يمكن الوصول إلى "My Cart" فقط للمستخدمين المسجلين
- كل مستخدم له عربة تسوق منفصلة في قاعدة البيانات
