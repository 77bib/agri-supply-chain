# نظام عربة التسوق المخصصة للعملاء - Customer Cart System

## نظرة عامة

تم تطوير نظام عربة تسوق مخصصة لكل عميل مع معلوماته الشخصية الكاملة، مما يوفر تجربة تسوق شخصية ومتخصصة لكل مستخدم.

## الميزات الرئيسية

### 👤 معلومات العميل الشخصية
- **البيانات الأساسية**: الاسم، البريد الإلكتروني، الهاتف
- **العنوان الكامل**: العنوان، المدينة، الولاية، الرمز البريدي، البلد
- **التفضيلات**: وقت التوصيل، تعليمات خاصة، طريقة الدفع المفضلة
- **التحديث المباشر**: إمكانية تعديل المعلومات في أي وقت

### 🛒 عربة التسوق المخصصة
- **عربة فريدة لكل عميل**: كل عميل له عربة تسوق خاصة به
- **حفظ تلقائي**: حفظ جميع التغييرات تلقائياً
- **إحصائيات مفصلة**: عدد العناصر، المجموع، آخر تحديث
- **ملاحظات خاصة**: إمكانية إضافة ملاحظات للطلب

### 💾 التخزين المتقدم
- **قاعدة بيانات MongoDB**: تخزين آمن وموثوق
- **معلومات شاملة**: جميع بيانات العميل والعربة
- **النسخ الاحتياطية**: نظام حماية متعدد المستويات

## هيكل البيانات

### نموذج عربة التسوق
```typescript
interface ICart {
  userId: string;                    // معرف المستخدم
  customerInfo: ICustomerInfo;       // معلومات العميل
  items: ICartProduct[];            // منتجات عربة التسوق
  totalAmount: number;              // المجموع الإجمالي
  itemCount: number;                // عدد العناصر
  lastUpdated: Date;                // آخر تحديث
  isActive: boolean;                // حالة النشاط
  notes: string;                    // ملاحظات
  savedForLater: ICartProduct[];    // محفوظ للاحقاً
}
```

### معلومات العميل
```typescript
interface ICustomerInfo {
  name: string;                     // الاسم الكامل
  email: string;                    // البريد الإلكتروني
  phone: string;                    // رقم الهاتف
  address: string;                  // العنوان
  city: string;                     // المدينة
  state: string;                    // الولاية
  zipCode: string;                  // الرمز البريدي
  country: string;                  // البلد
  preferences: {
    deliveryTime: string;           // وقت التوصيل المفضل
    specialInstructions: string;    // تعليمات خاصة
    preferredPaymentMethod: string; // طريقة الدفع المفضلة
  };
}
```

## الأمان والخصوصية

### حماية البيانات
- **التشفير**: جميع البيانات مشفرة في قاعدة البيانات
- **التحقق من الهوية**: جميع العمليات تتطلب توكن صالح
- **ملكية البيانات**: كل عميل يملك بياناته فقط

### التحكم في الوصول
- **التحقق من التوكن**: لكل طلب API
- **فصل البيانات**: كل عميل له بيانات منفصلة
- **حماية من SQL Injection**: استخدام Mongoose
