# نظام حفظ عربة التسوق - Cart Save System

## نظرة عامة

تم تطوير نظام شامل لحفظ واسترجاع عربة التسوق للعملاء، مما يضمن عدم فقدان المنتجات المختارة عند تسجيل الخروج أو مغادرة الموقع.

## الميزات الرئيسية

### 🔄 الحفظ التلقائي
- **حفظ عند التغيير**: يتم حفظ عربة التسوق تلقائياً عند إضافة أو إزالة أو تعديل المنتجات
- **حفظ عند الخروج**: حفظ تلقائي عند تسجيل الخروج
- **حفظ عند مغادرة الصفحة**: حفظ عند إغلاق المتصفح أو تغيير الصفحة
- **حفظ عند إخفاء الصفحة**: حفظ عند التبديل بين التطبيقات

### 💾 التخزين المزدوج
- **قاعدة البيانات**: التخزين الرئيسي في MongoDB
- **LocalStorage**: نسخة احتياطية محلية
- **الاسترجاع الذكي**: محاولة الاسترجاع من قاعدة البيانات أولاً، ثم من localStorage

### 🔔 الإشعارات الذكية
- **إشعار الاسترجاع**: إشعار عند العثور على عربة محفوظة
- **خيارات الاسترجاع**: إمكانية استرجاع أو تجاهل العربة المحفوظة
- **معلومات مفصلة**: عرض عدد العناصر والقيمة الإجمالية

## الملفات المضافة/المعدلة

### API Endpoints

### Components

### Store
- `lib/store.ts` - تحديث المتجر مع الحفظ التلقائي

## كيفية العمل

### 1. الحفظ التلقائي
```typescript
// عند إضافة منتج
addToCart(product, quantity) {
  // إضافة المنتج
  const newCart = [...cart, { product, quantity }]
  
  // حفظ تلقائي
  if (userLoggedIn) {
    saveCart(newCart) // قاعدة البيانات
    saveCartToLocalStorage(newCart) // localStorage
  }
}
```

### 2. الحفظ عند الخروج
```typescript
logoutUser() {
  // حفظ عربة التسوق قبل الخروج
  if (cart.length > 0) {
    await saveCart(cart)
    saveCartToLocalStorage(cart)
  }
  
  // تسجيل الخروج
  clearUserData()
}
```

### 3. الاسترجاع عند الدخول
```typescript
loginUser() {
  // تسجيل الدخول
  setUserData()
  
  // استرجاع عربة التسوق
  const savedCart = await loadCart()
  if (savedCart.length > 0) {
    const mergedCart = mergeCarts(savedCart, currentCart)
    setCart(mergedCart)
  }
}
```

## واجهة المستخدم

### صفحة المعلومات المحفوظة (`/saved-info`)
- عرض عربة التسوق الحالية
- عرض عربة التسوق المحفوظة
- أزرار الحفظ والاسترجاع والمسح
- معلومات عن آخر حفظ

### الإشعارات
- إشعار عند العثور على عربة محفوظة
- خيارات الاسترجاع أو التجاهل
- عرض تفاصيل العناصر والقيمة

## الأمان والخصوصية

### التشفير
- البيانات محفوظة بشكل آمن في قاعدة البيانات
- localStorage يستخدم للتخزين المحلي فقط

### التحقق من الهوية
- جميع عمليات الحفظ تتطلب توكن صالح
- التحقق من ملكية البيانات

### النسخ الاحتياطية
- نظام تخزين مزدوج (قاعدة البيانات + localStorage)
- استرجاع ذكي من المصدر المتاح

## الاستخدام

### للعملاء
1. **التسوق العادي**: إضافة منتجات للعربة
2. **الحفظ التلقائي**: لا حاجة لتدخل المستخدم
3. **الاسترجاع**: إشعار تلقائي عند العودة
4. **الإدارة اليدوية**: صفحة `/saved-info` للتحكم

### للمطورين
```typescript
// حفظ يدوي
import { saveCart } from '@/lib/cart-service'
await saveCart(cartItems)

// استرجاع يدوي
import { loadCart } from '@/lib/cart-service'
const savedCart = await loadCart()

// دمج عربات التسوق
import { mergeCarts } from '@/lib/cart-service'
const mergedCart = mergeCarts(savedCart, currentCart)
```
