# API للمنتجات - منصة سلسلة التوريد الزراعية

## الإعداد

### 1. تثبيت التبعيات
```bash
npm install mongoose --legacy-peer-deps
```

### 2. إعداد قاعدة البيانات
قم بإنشاء ملف `.env.local` في مجلد المشروع وأضف:
```
MONGODB_URI=mongodb://localhost:27017/agri-supply-chain
```

### 3. تشغيل MongoDB
تأكد من تشغيل MongoDB على جهازك أو استخدم MongoDB Atlas.

## استخدام API

### جلب جميع المنتجات (GET)
```bash
GET /api/products
```

**الاستجابة:**
```json
{
  "success": true,
  "count": 2,
  "data": [
    {
      "_id": "60f7b3b3b3b3b3b3b3b3b3b3",
      "name": "عصير برتقال طازج",
      "description": "عصير برتقال طبيعي 100%",
      "category": "عصائر",
      "price": 15.99,
      "quantity": 100,
      "supplier": "مزرعة الوادي الأخضر",
      "createdAt": "2024-01-15T10:30:00.000Z"
    }
  ]
}
```

### إنشاء منتج جديد (POST)
```bash
POST /api/products
Content-Type: application/json

{
  "name": "عصير برتقال طازج",
  "description": "عصير برتقال طبيعي 100%",
  "category": "عصائر",
  "price": 15.99,
  "quantity": 100,
  "supplier": "مزرعة الوادي الأخضر"
}
```

**الاستجابة الناجحة:**
```json
{
  "success": true,
  "message": "تم إنشاء المنتج بنجاح",
  "data": {
    "_id": "60f7b3b3b3b3b3b3b3b3b3b3",
    "name": "عصير برتقال طازج",
    "description": "عصير برتقال طبيعي 100%",
    "category": "عصائر",
    "price": 15.99,
    "quantity": 100,
    "supplier": "مزرعة الوادي الأخضر",
    "createdAt": "2024-01-15T10:30:00.000Z"
  }
}
```

**الاستجابة في حالة الخطأ:**
```json
{
  "success": false,
  "message": "الحقول المطلوبة مفقودة",
  "requiredFields": ["name", "category", "price", "quantity", "supplier"],
  "receivedData": {
    "name": "عصير برتقال"
  }
}
```

## الحقول المطلوبة

- **name** (مطلوب): اسم المنتج - نص
- **description** (اختياري): وصف المنتج - نص
- **category** (مطلوب): فئة المنتج - نص
- **price** (مطلوب): سعر المنتج - رقم موجب
- **quantity** (مطلوب): كمية المنتج - رقم موجب
- **supplier** (مطلوب): اسم المورد - نص

## رموز الحالة

- **200**: نجح الطلب (GET)
- **201**: تم إنشاء المنتج بنجاح (POST)
- **400**: خطأ في البيانات المرسلة
- **405**: طريقة طلب غير مسموح بها
- **500**: خطأ في الخادم
