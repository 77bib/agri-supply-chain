# نظام عربة التسوق المخصصة للعملاء - التوثيق الكامل

## نظرة عامة

تم تطوير نظام عربة تسوق مخصصة للعملاء يوفر لكل حساب قاعدة بيانات منفصلة لمعلومات عربة التسوق. هذا النظام يتيح للعملاء:

- حفظ معلومات عربة التسوق في قاعدة البيانات
- استعادة عربة التسوق عند تسجيل الدخول
- إدارة معلومات العميل الشخصية
- حفظ المنتجات للشراء لاحقاً
- إضافة ملاحظات خاصة

## الميزات الرئيسية

### 1. قاعدة بيانات منفصلة لكل عميل
- كل عميل له عربة تسوق خاصة في قاعدة البيانات
- البيانات محفوظة بشكل دائم ومرتبطة بـ `userId`
- استعادة تلقائية للبيانات عند تسجيل الدخول

### 2. إدارة معلومات العميل
- معلومات الاتصال (الاسم، البريد الإلكتروني، الهاتف)
- عنوان التوصيل الكامل
- تفضيلات التوصيل والدفع
- إمكانية التعديل والحفظ

### 3. إدارة المنتجات
- إضافة منتجات إلى عربة التسوق
- تعديل الكميات
- إزالة منتجات
- حفظ منتجات للشراء لاحقاً

### 4. الحفظ التلقائي
- حفظ تلقائي عند إغلاق المتصفح
- حفظ عند تغيير الصفحة
- حفظ في localStorage كنسخة احتياطية

## كيفية الاستخدام

### للعملاء:

1. **تسجيل الدخول**: قم بتسجيل الدخول إلى حسابك
2. **الوصول إلى عربة التسوق**: انقر على "My Cart" في القائمة المنسدلة
3. **إضافة منتجات**: تصفح المنتجات وأضفها إلى عربة التسوق
4. **إدارة المعلومات**: عدّل معلوماتك الشخصية وتفضيلات التوصيل
5. **الحفظ التلقائي**: البيانات تُحفظ تلقائياً عند أي تغيير

### للمطورين:

#### إنشاء عربة تسوق جديدة:
```typescript
import { createCustomerCart } from '@/lib/customer-cart-service'

const customerInfo = {
  name: "اسم العميل",
  email: "email@example.com",
  phone: "+212123456789",
  address: "العنوان",
  city: "المدينة",
  state: "الولاية",
  zipCode: "12345",
  country: "Morocco",
  preferences: {
    deliveryTime: "anytime",
    specialInstructions: "",
    preferredPaymentMethod: "card"
  }
}

const result = await createCustomerCart(userId, customerInfo)
```

#### إضافة منتج إلى عربة التسوق:
```typescript
import { addItemToCustomerCart } from '@/lib/customer-cart-service'

const product = {
  productId: "product123",
  name: "عصير برتقال طازج",
  description: "عصير برتقال طبيعي 100%",

  category: "عصائر",

  farmer: "مزرعة الأخضر",
  certifications: ["عضوي", "طبيعي"],
  quantity: 2
}

const result = await addItemToCustomerCart(userId, product)
```

## هيكل قاعدة البيانات

### نموذج عربة التسوق (Cart Schema):
```typescript
interface ICart {
  userId: string;                    // معرف المستخدم
  customerInfo: ICustomerInfo;       // معلومات العميل
  items: ICartProduct[];            // المنتجات في عربة التسوق
  totalAmount: number;              // المجموع الكلي
  itemCount: number;                // عدد المنتجات
  lastUpdated: Date;                // آخر تحديث
  isActive: boolean;                // حالة النشاط
  notes: string;                    // ملاحظات خاصة
  savedForLater: ICartProduct[];    // منتجات محفوظة للشراء لاحقاً
  createdAt: Date;                  // تاريخ الإنشاء
  updatedAt: Date;                  // تاريخ التحديث
}
```

## API Endpoints

### GET /api/customer-cart
- **الوصف**: جلب عربة التسوق للعميل
- **المصادقة**: مطلوبة (JWT Token)
- **الاستجابة**: بيانات عربة التسوق أو إنشاء عربة جديدة

### POST /api/customer-cart
- **الوصف**: إنشاء عربة تسوق جديدة
- **المصادقة**: مطلوبة (JWT Token)
- **البيانات**: customerInfo
- **الاستجابة**: تأكيد الإنشاء

### PUT /api/customer-cart
- **الوصف**: تحديث معلومات العميل
- **المصادقة**: مطلوبة (JWT Token)
- **البيانات**: customerInfo
- **الاستجابة**: تأكيد التحديث

### POST /api/customer-cart/items
- **الوصف**: إضافة منتج إلى عربة التسوق
- **المصادقة**: مطلوبة (JWT Token)
- **البيانات**: product
- **الاستجابة**: تأكيد الإضافة

### PUT /api/customer-cart/items
- **الوصف**: تحديث كمية منتج
- **المصادقة**: مطلوبة (JWT Token)
- **البيانات**: { productId, quantity }
- **الاستجابة**: تأكيد التحديث

### DELETE /api/customer-cart/items
- **الوصف**: إزالة منتج أو تفريغ عربة التسوق
- **المصادقة**: مطلوبة (JWT Token)
- **البيانات**: { productId } أو { clearAll: true }
- **الاستجابة**: تأكيد الإزالة

## الأمان والتحقق

### التحقق من التوكن:
- جميع API endpoints تتطلب JWT token صالح
- التحقق من صحة التوكن قبل أي عملية
- رفض الطلبات غير المصرح بها

### حماية البيانات:
- كل عميل يمكنه الوصول فقط لبياناته
- التحقق من `userId` في كل عملية
- تشفير البيانات الحساسة
